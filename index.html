Ecosystem Dev App

Ecosystem Dev App helps authorized internal users review Google Ads account performance, recommendations, and landing page optimization context.

Purpose

This application is designed to support Google Ads operations review. It uses Google OAuth and the Google Ads API to display account information, campaign performance metrics, and recommendations for accounts that the authenticated user is authorized to access.

Current Scope

  • Read accessible Google Ads customer accounts.
  • Display campaign performance metrics when API access is available.
  • Display Google Ads recommendations for human review.
  • Support landing page optimization analysis in a local/internal workflow.

Access

Access is limited to authorized internal users. The application is not intended for use by the general public. Users must authenticate with a Google account that has permission to access the relevant Google Ads account or manager account.

Data Handling

Google Ads data is used only to display operational insights to authorized users. The application does not sell user data and does not share Google Ads data with third parties.
