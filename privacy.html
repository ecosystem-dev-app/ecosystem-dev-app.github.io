Privacy Policy

Last updated: June 19, 2026

Overview

Ecosystem Dev App is an internal Google Ads operations support tool. This Privacy Policy explains how the application uses information obtained through Google OAuth and the Google Ads API.

Information We Access

With user authorization, the application may access Google Ads account information that the authenticated user is permitted to view, including customer identifiers, campaign performance metrics, and Google Ads recommendations.

How We Use Information

Information is used to display account performance, review recommendations, and support landing page optimization analysis for authorized internal users.

Data Sharing

The application does not sell Google user data. Google Ads data is not shared with third parties except when required by law or with explicit authorization from the account owner.

Data Storage

The application is currently operated as a local/internal tool. OAuth tokens and API configuration are stored locally for development and internal operation. Users should protect their local environment and credentials according to their security requirements.

Data Retention and Deletion

Users may revoke application access at any time from their Google Account security settings. Local credentials or tokens can be deleted from the local application environment by the operator.

Google API Services User Data Policy

The application's use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Contact

For privacy questions, contact the developer using the support email configured on the OAuth consent screen.
